Quick Installation

# Clone repository
git clone https://github.com/nitindahiya-dev/terminalAI.git

# Start AI backend
docker run -p 5500:5500 d0ckmg/free-gpt4-web-api

# Build and run
cargo build --release
cargo run -- --interactive
